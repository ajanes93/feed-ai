# Update PR Template

View or update the PR body template used by the `/pr` command.

## Instructions

1. Read the current template from `.claude/pr-template.md`
2. Show the user the current template
3. If the user provided instructions via `$ARGUMENTS`, update the template accordingly
4. Otherwise, ask the user what they'd like to change
5. Write the updated template to `.claude/pr-template.md`
6. Confirm the change
